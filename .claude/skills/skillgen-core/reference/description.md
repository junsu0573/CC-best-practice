# Description recipe (discovery-critical)

## Required properties

- third-person
- includes WHAT + WHEN + key terms
- <= 1024 chars

## Recommended one-liner format

- `Does <WHAT>. Use when <WHEN/TRIGGERS>. Avoid when <AVOID>.`

## Examples

- Good:
  - `Extracts text and tables from PDFs. Use when working with PDF files, forms, or document extraction.`
- Bad:
  - `I can help you with PDFs.`
  - `Helps with documents.`

## Trigger terms

Include words users actually say:

- file types: pdf, docx, xlsx
- actions: migrate, deploy, lint, test, refactor, generate
- context: PR, CI, regression, rollout, rollback
